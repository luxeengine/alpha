Draw

luxe.Draw

The draw API class implementation. This class handles quick access to drawing common geometry options, and is a facilitator for higher level drawing. These can be used given to a Visual to convert them into a higher level entity for use with the default scene.

arc
arc(options:luxe.options.DrawArcOptions) : phoenix.geometry.ArcGeometry
Draw an arc (open circle) at the specified x,y, between start_angle and end_angle with radius r (or rx,ry optionally for an ellipsical arc), with options

    //draws a 60 degree arc center screen
var geometry = Luxe.draw.arc({
    r : 70,
    end_angle:60,
    x : Luxe.screen.w/2,
    y : Luxe.screen.h/2,
    color : new Color(1,1,1,1)
});

poly

poly(options:luxe.options.DrawPolygonOptions) : phoenix.geometry.Geometry
Draw an arbitrary polygon of points with options. Uses triangle fans (no additional verts added) for solid, Uses lines for non-solid, adding verts as needed to form lines.

    //draw a list of points as a polygon
Luxe.draw.poly({
    solid : true,
    color: new Color().rgb(0xff4b03),
    points : [
        new Vector(28, 114),
        new Vector(77, 104),
        new Vector(84, 109),
        new Vector(108, 184),
        new Vector(44, 202),
        new Vector(25, 120),
        new Vector(28, 114),
    ]
});

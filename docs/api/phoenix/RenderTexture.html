bind>
bind() : Void
Bind this texture to the it’s active texture slot, and it’s texture id to the texture type. Calling this repeatedly is fine, as the state is tracked by the RenderState.

fetch>

fetch(_into:snow.api.buffers.Uint8Array, _x:Int, _y:Int, _w:Int, _h:Int) : snow.api.buffers.Uint8Array
Fetch the pixels from the texture id, storing them in the provided array buffer view. Returns image pixels in RGBA format, as unsigned byte (0-255) values only. This means that the view must be w * h * 4 in length, minimum. By default, x and y are 0, 0, and the texture width and height are used (not width_actual/height_actual)
